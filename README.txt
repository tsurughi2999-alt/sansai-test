==============================================
 🌿 山菜採り手帳 PWA — 配布・インストール手順
==============================================

【ファイル構成】
  index.html   ... メインアプリ
  manifest.json ... PWA設定
  sw.js         ... オフライン対応(Service Worker)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【ホスティング方法（どれか1つ選ぶ）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

★ A) GitHub Pages（無料・おすすめ）
  1. GitHubアカウント作成（無料）
  2. 新しいリポジトリを作成
  3. 3ファイルをアップロード
  4. Settings → Pages → Branch: main → Save
  5. https://ユーザー名.github.io/リポジトリ名/ でアクセス可能

★ B) Netlify Drop（最速・ドラッグ&ドロップ）
  1. https://app.netlify.com/drop を開く
  2. 3ファイルをフォルダにまとめてドラッグ&ドロップ
  3. 自動でURLが発行される（例: https://abc123.netlify.app）

★ C) Firebase Hosting（Googleアカウントで利用可）
  npm install -g firebase-tools
  firebase login
  firebase init hosting
  firebase deploy

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【Androidにインストールする手順】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. ChromeでURLを開く
  2. 画面下部に「ホーム画面に追加」バナーが表示される
     ※表示されない場合: メニュー(⋮) →「アプリをインストール」
  3. 「インストール」をタップ
  4. ホーム画面に「山菜手帳」アイコンが追加される ✅

【iPhoneにインストールする手順】
  1. SafariでURLを開く
  2. 共有ボタン(□↑) →「ホーム画面に追加」
  3. 「追加」をタップ ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【友人への共有方法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  URLをLINE・メッセージで送るだけ！

【オフライン対応】
  一度開くと次回からオフラインでも使えます。
  ただし天気予報はネット接続が必要です。
